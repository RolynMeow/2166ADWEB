Text Format Pipes

  • LOWERCASE NAME: {{ myName | lowercase }}
  • UPPERCASE NAME: {{ myName | uppercase }}
  • DATE UPPERCASE: {{ presentDate | date: 'fullDate' | uppercase }}

Date Pipes

  • TIME: {{ presentDate | date:'shortTime' }}
  • DATE WITH TIME: {{ presentDate | date:"MMM dd, yyyy 'at' hh:mm a" }}
  • DATE: {{ presentDate | date }}

Currency Pipes

  • PHP - DOLLAR: {{ pesos | currency:'PHP':true }} ==> {{ dollar| currency:'':true }}
  • JEEPNEY FARE: {{ jeep | currency:'PHP':true }}
  • JEEPNEY FARE IN DOLLAR: {{ jeep | currency:'PHP' }} ==> {{ jeepDollar | currency:'' }}

Decimal Pipes

  • 23.32451 - 3 DECIMAL: {{ 23.32451 | number:'1.3' }}
  • 69.9 WHOLE NUMBER: {{ 69.9 | number:'1.0-0' }}
  • 99% TO WHOLE NUMBER: {{ .99 | number:'1.2' }}

JSON Pipes

  • {{ data | json }}

Percent Pipes

  • 0.69 PERCENTAGE: {{ a | percent }}
  • 0.25 PERCENTAGE: {{ b | percent:'1.2' }}
  • 0.50 PERCENTAGE: {{ c | percent:'2.1' }}

Slice Pipes

  • FIRST COLOR: {{ Color | slice:0:1 }}
  • 3 LAST COLORS: {{ Color | slice:2 }}
  • LAST COLOR: {{ Color | slice:4 }}
